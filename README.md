# ez-starting
更快、更简洁的edge浏览器起始页

# 使用

> 在打包之前，请确保你的电脑上安装了`nodejs`.

1. 进入命令行，执行命令
```
yarn install
yarn run build
```
2. 进入edge/chrome/360的插件管理界面，打开开发者模式
3. 选择`加载已解压的拓展程序`，选择项目目录下的`dist`文件夹
4. 重启浏览器
